## Requirements
- `Flutter >3.7.6`
- `Android Emulator`
  - Built with `Pixel 6 Android 11`
- `Firebase CLI`
- `Flutterfire`
  - We used a private firebase project. Would need to setup a seperate firebase project.
  
  
 ## Instructions
 - Clone the repo
 - `flutter pub get`
 - Start the android emulator
 - `flutter run`
